# WebStudio 🌐

**WebStudio** is a simple, modern, and fully responsive landing page for a digital agency. It presents the company's services, team, and portfolio. 

🔗 **[Live Demo](https://ibuizle.github.io/Webstudio/)**

## 📖 About the Project

This is a practice project for building web pages. The site adapts to different screen sizes and includes the following sections:
- **Header:** Navigation menu and contact information.
- **Hero:** Main banner with an "Order Service" button that opens a contact form.
- **Features:** Four main reasons to work with the company.
- **Team:** Cards showing team members and their social media links.
- **Portfolio:** A gallery of projects with nice hover effects.
- **Footer:** Logo, contact info, and an email subscription form.

## 🛠 Built With

- **HTML5:** For the structure of the page.
- **CSS3:** For styling, including Flexbox and hover effects.
- **Responsive Web Design:** Looks great and works well on mobile phones, tablets, and desktops.
- **JavaScript:** A small script to open and close the modal window (contact form).
